# Salesforce MCP Server Integration with Claude Code & Claude Desktop

This repository contains the architecture, configuration steps, and security workflows for integrating **Salesforce-Hosted Model Context Protocol (MCP) Servers** with **Claude Code (CLI)** and **Claude Desktop**. 

By utilizing **Headless 360 architectures**, this integration enables Claude to natively fetch, aggregate, and interact with live CRM data directly through terminal commands or chat interfaces without opening a browser window.

---

## 🏗️ Architecture Overview

The system uses the Model Context Protocol (MCP) as a secure bridge between Anthropic's Claude models and Salesforce Orgs. Data governance is strictly enforced via Salesforce External Client Apps (ECA) and runtime user-permission prompts.
